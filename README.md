# Banking Microservices Application

A real-world banking backend system built using Spring Boot and Microservices architecture.

## Microservices
- Account Service – Account creation & management
- Customer Service – Customer profile management
- Loan Service – Loan processing & eligibility

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- REST APIs
- MySQL / PostgreSQL
- Docker (planned)
- Kubernetes (planned)

## Architecture
- Independent microservices
- REST-based communication
- Centralized configuration (upcoming)
